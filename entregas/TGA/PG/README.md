### Para executar o jogo, basta clonar o repositório e abrir o arquivo PG.sln com o visual studio, e executar no VS na função de Debug x64.

## Controles
### A qualquer momento
- ESC: Fechar jogo
### Menu Inicial
- AD: Trocar níveis 
- Espaço: Iniciar jogo
### Jogo
- WASD: Movimentação
- Espaço: Atirar adaga
### Tela de vitória/derrota
- Espaço: Retornar ao menu
